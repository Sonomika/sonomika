# Sonomika

**VJ software with an ability to make AI effects and sources. Built with web technology and packaged with Electron**

Sonomika is a free real time visual performance tool for live shows, streaming and installations. It is designed for VJs, DJs and visual artists.

## About

This beta release is built for Windows. It should compile on macOS, but this is unverified. Help with macOS builds is welcome.

## Key Features

- **AI effect generation** and AI compatible workflow
- **Column and timeline modes**
- **Media playback** for video, audio and Three.js
- **Full MIDI mapping** including focus control
- **More than forty built in effects**
- **Multi layer composition** and BPM sync
- **Scene presets** with smooth transitions
- **Internal audio recording**
- **Media library** with drag and drop import
- **Generative sequencing**
- **WebM video recording**
- **Live code editing** and custom AI templates
